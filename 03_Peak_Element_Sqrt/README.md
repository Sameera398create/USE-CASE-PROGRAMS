 Experiment 3: Peak Element & Square Root

##  Objective
To find a peak element in an array and compute square root efficiently.

## 📖 Background
Peak element is an element greater than its neighbors. Binary search helps in efficient computation.

##  Methodology
1. Traverse array to find peak element
2. Use mathematical or binary method for square root
3. Return results

##  Input
Array elements and number

## Output
Peak element and square root value

##  Complexity Analysis
- Time Complexity: O(log n)
- Space Complexity: O(1)

## Conclusion
The program efficiently finds peak element and computes square root.
