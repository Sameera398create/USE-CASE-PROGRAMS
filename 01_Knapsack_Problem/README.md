 Experiment 1: Knapsack Problem

## Objective
To implement the Knapsack Problem and maximize profit within a given capacity.

## Background
The Knapsack Problem is a classic optimization problem where items are selected based on weight and value constraints.

## Methodology
1. Define weights and values
2. Set capacity
3. Apply dynamic programming
4. Compute maximum profit

##  Implementation Details
- Uses arrays for weights and values
- Applies DP table approach
- Tracks optimal selection


## Input
Weights, Values, Capacity

## Output
Maximum Profit


##  Complexity Analysis
- Time Complexity: O(nW)
- Space Complexity: O(n)
  
## Conclusion
The program successfully finds the optimal solution.
